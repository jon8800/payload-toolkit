    Create `src/hooks/revalidatePage.ts`:
    - Export `revalidatePage` as `CollectionAfterChangeHook`
    - Guard with `if (!context.disableRevalidate)`
    - If `doc._status === 'published'`: revalidatePath for the page (slug 'home' -> '/', else `/${doc.slug}`), plus revalidateTag('pages-sitemap')
    - If previousDoc was published and doc is now not published: revalidatePath old path, revalidateTag('pages-sitemap')
    - Log via `payload.logger.info`
    - Return doc

    Create `src/hooks/revalidatePost.ts`:
    - Same pattern as revalidatePage but paths use `/blog/${doc.slug}` prefix
    - Also revalidatePath('/blog') to refresh the archive page
    - Use revalidateTag('posts-sitemap')
    - Return doc

    Create `src/hooks/revalidateTemplatePart.ts`:
    - Same guard pattern
    - If published: `revalidatePath('/', 'layout')` to refresh all pages (template parts affect layout)
    - Also revalidateTag('pages-sitemap') and revalidateTag('posts-sitemap')
    - If unpublished: same `revalidatePath('/', 'layout')`
    - Return doc

    Create `src/app/(frontend)/next/preview/route.ts`:
    - Export async GET handler
    - Parse searchParams for path, collection, slug, previewSecret
    - Validate previewSecret against process.env.PREVIEW_SECRET (return 403 if mismatch)
    - Validate path, collection, slug exist (return 404 if missing)
    - Validate path starts with '/' (return 500 if not)
    - Authenticate user via `payload.auth({ req: req as unknown as PayloadRequest, headers: req.headers })`
    - If no user, disable draft mode and return 403
    - Enable draft mode via `(await draftMode()).enable()`
    - Redirect to path
    - Follow exact pattern from 04-RESEARCH.md Pattern 1

    Create `src/app/(frontend)/next/exit-preview/route.ts`:
    - Export async GET handler
    - `(await draftMode()).disable()`
    - Return Response('Draft mode is disabled')
