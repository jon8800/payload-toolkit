  <action>
1. Install dependencies in the starter app:
   ```bash
   cd apps/starter && pnpm add culori google-font-metadata && pnpm add -D @types/culori
   ```

2. Create `apps/starter/src/lib/themeUtils.ts` with:
   - `hexToOklch(hex: string): string` — uses culori/fn with tree-shaking (import parse, formatCss, converter, useMode, modeOklch, modeRgb from 'culori/fn'). Returns formatted oklch CSS string. Falls back to input hex if parse fails.
   - `buildCSSVariables(theme: ThemeData): Record<string, string>` — takes the full theme object (colors.derivedTokens + spacing + borderRadius + fonts), builds a flat Record of CSS variable names to values. Only includes variables where the admin has set a value (skip nulls/undefined). For spacing: compute `--spacing-xs` through `--spacing-2xl` from `baseMultiplier` (xs=baseMultiplier*1, sm=baseMultiplier*2, md=baseMultiplier*4, lg=baseMultiplier*6, xl=baseMultiplier*8, 2xl=baseMultiplier*12, output as `${value}px`). For borderRadius: output `--radius: {value}rem`. For fonts: output `--font-sans: '{fontFamily}', sans-serif` and `--font-mono: '{monoFamily}', monospace`.
   - Export a `ThemeData` type representing the shape of the ThemeSettings global document.

3. Create `apps/starter/src/fields/theme/deriveColors.ts` with:
   - `deriveAllColors(core)` — takes core color inputs (primary, secondary, accent, muted, destructive, background, foreground as hex strings), converts each to oklch via hexToOklch, then derives:
     - All direct conversions: --primary, --secondary, --muted, --accent, --destructive, --background, --foreground
     - Auto-computed foregrounds: for each semantic color, compute foreground by checking oklch lightness (L > 0.5 = dark foreground oklch(0.145 0 0), else light foreground oklch(0.985 0 0))
     - Derived from background: --card = background, --card-foreground = foreground, --popover = background, --popover-foreground = foreground
     - Derived from secondary: --border = desaturated secondary (multiply chroma by 0.3), --input = same as border
     - --ring = primary with lightness shifted +0.2
     - Sidebar vars: --sidebar = background with lightness -0.015, sidebar-foreground = foreground, sidebar-primary = primary, sidebar-accent = accent, sidebar-border = same as border, sidebar-ring = same as ring
     - Chart colors: rotate primary hue by 0, 30, 60, 90, 120 degrees for chart-1 through chart-5, varying lightness slightly
   - Helper functions: `adjustLightness(hex, deltaL)`, `desaturate(hex, factor)`, `rotateHue(hex, degrees)` — all using culori/fn
   - Return type: `Record<string, string>` where keys are CSS variable names WITHOUT the -- prefix (e.g., 'primary', 'card-foreground') and values are oklch strings

Note: Use culori/fn imports for tree-shaking (NOT the main 'culori' entry). Register only modeOklch and modeRgb via useMode.
